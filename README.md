# HTP Test Analyzer

A tool for analyzing HTP tests.

## Features

- Parse and analyze HTP test logs
- Generate reports

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Arihant25/htp-test-analyzer.git
   cd htp-test-analyzer
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Run the project:
   ```bash
    npm run dev
```
